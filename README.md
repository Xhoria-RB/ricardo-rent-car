# Rent a Car Proyect

This is an app about renting cars. More details coming soon.

## Instructions
Use the followings
```bash
# To install the dependencies
npm install
cd client && npm install

# To run server & client (5000 & 3000)
npm run dev

# To run server only
npm run server

# To run client only
npm run client

#Build for production
cd client && npm run build

#Graphiql http://localhost:5000/graphql 
```
## App details:
### Technologies:
Back End (server):
 - Node.Js
 - Express.Js
 

Front-End (client): 
 - React.Js
 - React-Router
